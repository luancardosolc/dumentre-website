# dumentre-website

Website oficial da [Dumentre](https://dumentre.com) — arquitetos de sistemas autônomos.

> *"Connecting the world that exists to the systems that should exist."*

## Stack

- **Next.js 15** (App Router)
- **TypeScript**
- **Tailwind CSS v4**
- **shadcn/ui**
- **Vercel** (deploy)
